Last Post : Forum will be sorted with the topic that has the most recent activity at the top. The most recent activity could be either the newest created discussion topic or the discussion topic with the most recent reply.

First Name : Sorts discussion topics alphabetically by the user's first name.

Last Name : Sorts discussion topics alphabetically by the user's last name

Oldest Creation date: Forum will be sorted with the oldest created topics at the top descending chronologically.

Newest creation date: Forum will be sorted with the newest created topics at the top and will descend chronologically.
